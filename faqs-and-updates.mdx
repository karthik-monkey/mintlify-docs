## Frequently Asked Questions

### General

#### What is Aida?

Aida is an AI-powered platform that helps developers integrate intelligent assistance into their applications. Our tools make it easy to add contextual help, user guidance, and support automation to any software product.

#### How do I get started?

Check out our [Quickstart](/quickstart) guide for step-by-step instructions on setting up your first project.

### Pricing

#### Do you offer a free tier?

Yes! We offer a generous free tier that includes:
- Up to 1,000 API calls per month
- Basic analytics
- Access to core features

#### How does billing work?

Billing is based on API usage and is calculated monthly. You'll only be charged for what you use beyond the free tier limits.

### Technical

#### Which platforms do you support?

Our SDKs currently support:
- JavaScript/TypeScript (React, Vue, Angular)
- Python
- Ruby
- Java

#### Is my data secure?

Absolutely. We take security seriously and implement industry-standard encryption and access controls. For more details, please review our [Privacy Policy](/privacy-policy).

#### How can I get help if I'm stuck?

We offer several support channels:
- Email support at [info@aida.vc](mailto:info@aida.vc)
- Community forums
- Detailed documentation

For enterprise customers, we also offer dedicated support with guaranteed response times.
